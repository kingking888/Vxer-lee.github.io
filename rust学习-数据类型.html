整型

整数 是一个没有小数部分的数字。我们在第二章使用过 u32 整数类型。该类型声明表明，它关联的值应该是一个占据 32 比特位的无符号整数（有符号整数类型以 i 开头而不是 u）。表格 3-1 展示了 Rust 内建的整数类型。在有符号列和无符号列中的每一个变体（例如，i16）都可以用来声明整数值的类型。

表格 3-1: Rust 中的整型

长度 有符号 无符号
8-bit（byte）1byte i8 u8
16-bit（word）2byte i16 u16
32-bit（int） 4byte i32 u32
64-bit（int64）8byte i64 u64
128-bit i128 u128
arch(32、64int) isize usize

可以使用表格 3-2 中的任何一种形式编写数字字面值。注意除 byte 以外的所有数字字面值允许使用类型后缀，例如 57u8，同时也允许使用 _ 做为分隔符以方便读数，例如1_000。

表格 3-2: Rust 中的整型字面值

数字字面值 例子
Decimal (十进制) 2020_09_20
Hex (十六进制) 0xff
Octal (八进制) 0o77
Binary (二进制) 0b1111_0000
Byte (单字节字符)(仅限于u8) b'A'
